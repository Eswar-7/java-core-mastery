# Java Basics Interview Questions


## 1. What is Java?

**Answer:**
Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems (now owned by Oracle). Java programs run on any system that has a JVM (Java Virtual Machine).


## 2. Why is Java platform independent?

**Answer:**
Java is platform independent because Java source code is compiled into bytecode, which runs on the Java Virtual Machine (JVM). The JVM is available for different operating systems.

## 3. What is JVM?

**Answer:**
JVM (Java Virtual Machine) is a virtual machine that executes Java bytecode. It converts bytecode into machine code and provides platform independence.


## 4. What is JDK?

**Answer:**
JDK (Java Development Kit) is a software package used to develop Java applications. It includes the JRE, compiler (javac), debugger, and other development tools.


## 5. What is JRE?

**Answer:**
JRE (Java Runtime Environment) provides the libraries and JVM required to run Java applications. It does not contain development tools like the compiler.


## 6. Difference between JDK, JRE and JVM?

**Answer:**

| JDK | JRE | JVM |
| Used to develop Java programs | Used to run Java programs | Executes Java bytecode |
| Contains JRE | Contains JVM | Converts bytecode to machine code |


## 7. What is bytecode?

**Answer:**
Bytecode is the intermediate code generated by the Java compiler. It is executed by the JVm

## 9. What are variables?

**Answer:**
Variables are named memory locations used to store data values during program execution.

Example:

```java
int age = 21;
```

## 10. What are primitive data types in Java?

**Answer:**
Java has 8 primitive data types:

- byte
- short
- int
- long
- float
- double
- char
- boolean

## 11. Difference between int and long?

**Answer:**

- int stores 32-bit integer values.
- long stores 64-bit integer values.


## 12. Difference between float and double?

**Answer:**

- float occupies 4 bytes and is less precise.
- double occupies 8 bytes and is more precise.

## 13. What is type casting?

**Answer:**
Type casting is the process of converting one data type into another.

There are two types:

- Implicit (Widening)
- Explicit (Narrowing)

## 14. What is implicit type casting?

**Answer:**
Implicit type casting is performed automatically by Java when converting a smaller data type into a larger data type.

Example:

```java
int num = 100;
double value = num;
```

## 15. What is explicit type casting?

**Answer:**
Explicit type casting is done manually by the programmer when converting a larger data type into a smaller one.

Example:

```java
double price = 99.99;
int amount = (int) price;
```

## 16. What are operators?

**Answer:**
Operators are symbols used to perform operations on variables and values.

Examples:

- Arithmetic
- Relational
- Logical
- Assignment
- Increment/Decrement


## 17. What is the difference between == and = ?

**Answer:**

- = is the assignment operator.
- == is the comparison operator.

## 18. What is Scanner class?

**Answer:**
Scanner is a class in the java.util package used to take input from the user.

Example:

```java
Scanner sc = new Scanner(System.in);
```

## 19. What is System.out.println()?

**Answer:**
System.out.println() is used to print output on the console and move the cursor to the next line.


## 20. Why do we close the Scanner object?

**Answer:**
Closing the Scanner object releases system resources and prevents resource leaks.

Example:

```java
sc.close();
```
